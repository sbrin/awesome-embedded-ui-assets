# Awesome Embedded UI Assets

This repository contains a collection of icons and fonts suitable for embedded electronics GUI design.

## Sources

The assets in this repository are sourced from the following projects:

- [Lopaka](https://github.com/sbrin/lopaka)
- [u8g2](https://github.com/olikraus/u8g2)
- [Adafruit-GFX-Library](https://github.com/adafruit/Adafruit-GFX-Library)
- [GUIslice-Builder](https://github.com/ImpulseAdventure/GUIslice-Builder)
- [MaterialDesign](https://github.com/Templarian/MaterialDesign)
- [flipperzero-firmware](https://github.com/flipperdevices/flipperzero-firmware)

## GUI Design Toolbox
These and many other assets are available in the [Lopaka](https://lopaka.app) ebmedded GUI design toolbox.
